# Cloning

The project contains external libraries, to clone them too use:
`git clone --recursive https://github.com/erdUha/design-endterm.git`

# Requirements

Cmake; Ninja

# Setup

Create the directory `build/` if not exists and execute `cmake -B build -G Ninja`, then `./build.sh` to build the project. Then execute the needed part by executing the executable, for example `./build/PartA`
